# UpTask

Project and task management

* Features:
  * Authentication
  * Roles Assignments
    * Manager
    * Collaborator
  * Project CRUD
  * Task CRUD
  * Notes CRUD

**CRUD:** Create Read Update Delete

* Technologies:
  * React
  * TypeScript
  * Vite
  * TanStack Query
  * Zustand
  * Zod
